# Open All Treasure Pods

A Slime Rancher 2 mod that automatically opens all treasure pods.

## Installation

1. Install [Melon Loader](https://melonwiki.xyz/#/?id=automated-installation) for Slime Rancher 2
1. Download the [latest release](https://github.com/markekraus/SR2OpenAllTreasurePods/releases)
1. Copy `OpenAllTreasurePods.dll` to the `Mods` folder in your Slime Rancher 2 install folder (i.e. `C:\Program Files (x86)\Steam\steamapps\common\Slime Rancher 2\Mods`)
